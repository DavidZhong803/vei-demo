# VEI — One‑Page Product Description

## One‑liner

**Upload Your Value. AI Continuously Finds and Executes Your Best Value Path.**
上传你的价值主体，AI 持续寻找并执行你的最佳价值实现路径。

## What VEI is (and is not)

| VEI is NOT | VEI IS |
| --- | --- |
| A traditional internet product | An **AI‑native agent** |
| A consulting / advisory system | A system that **acts**, not just advises |
| A one‑time analysis report | A **continuously working** agent that updates over time |

## The product in three stages

1. **Upload the value subject**
   Users upload anything that represents their value: founding team, business
   plan (PDF/PPT), website, GitHub, technical docs, patents, product intro,
   financials, company profile. No complex forms — the AI starts on its own.

2. **AI Working (autonomous)**
   Not a chat, not a survey. The page shows the agent:
   understanding the value subject → team capability → technology assets →
   business model → industry → competition → the current time window →
   searching for the best value path → generating an execution plan.

3. **AI Execution (continuous)**
   The output is action, not just analysis. For each case the agent returns:
   - Current value subject
   - Best value path
   - Recommended time window
   - Recommended business model
   - Recommended actions
   Then **Continuous Optimization** keeps monitoring market, technology,
   business‑model, competition and capital changes — and updates the best value
   path whenever a better one emerges.

## Backend reasoning framework (illustrative only in V0.1)

```
Time → Event → State → Position → Value → Evolution → Advantage
```

This belongs to the backend reasoning engine. V0.1 only **visualizes** it as an
animation/flow; it does not perform real inference.

## Demo cases (all mock data)

- **Meta** — global social + AI infrastructure platform
- **Marvell** — data‑center custom silicon & interconnect
- **AI Startup** — simulated early‑stage AI company

## Design principles

AI‑native · minimal · premium · Apple/OpenAI style · dark background · generous
whitespace · no dashboards · no admin panels. The focus is always: **the AI is
continuously working.**

## Acceptance

If a first‑time user understands, within 3 minutes, that VEI is a continuously
working AI agent — uploading a value subject, understanding it automatically,
continuously finding the best value path, and continuously executing value
optimization — V0.1 is accepted.
